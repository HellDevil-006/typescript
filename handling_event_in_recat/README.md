# Handling Events in React

This repository contains examples and demonstrations of handling events in React applications.

## Project Structure

This repository contains three separate React applications:

1. `event` - Basic event handling examples
2. `handling_events` - Advanced event handling techniques
3. `passing` - Passing data with events

## Getting Started

To run any of these applications:

1. Navigate to the desired directory:
   ```
   cd event
   # or
   cd handling_events
   # or
   cd passing
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

## Applications Overview

### Event
Basic examples of handling user interactions like clicks, form submissions, and keyboard events.

### Handling Events
Advanced techniques for event handling including custom hooks, event delegation, and performance optimization.

### Passing
Examples showing how to pass data between components using events and callbacks.

## Prerequisites

- Node.js (version 14 or higher)
- npm (usually comes with Node.js)
